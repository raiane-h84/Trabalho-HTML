✅ A História da Internet

A história começou para desenvolver um meio de comunicação, onde mesmo distantes as pessoas pudessem se comunicar. Em 1835, ocorreu a criação do telégrafo, até a popularização da rede (1997) e a instalação de cabos submarinos (2004).

Destaca marcos como a criação do ENIAC (1945), ARPANET (1969), e-mail (1972) e o protocolo TCP/IP (1973).

Também aborda os principais protocolos de comunicação, a evolução dos navegadores e a importância dos domínios e URLs.

✅ Protocolos da Internet

Os protocolos são regras que definem como os dispositivos na internet se comunicam. Exemplos incluem FTP (para transferência de arquivos), HTTP/HTTPS (para navegação web) e TCP/IP (para transmissão de dados).

✅ Protocolos de E-mail

Os e-mails utilizam protocolos para envio e recebimento de mensagens. O SMTP é responsável pelo envio, enquanto o POP3 e o IMAP são usados para receber mensagens, com o IMAP permitindo sincronização entre dispositivos.

✅ Protocolo de Transferência de Hipertexto (HTTP)

O HTTP define regras para a troca de informações na web, como páginas HTML, imagens e vídeos. O HTTPS é uma versão segura que usa criptografia para proteger os dados transmitidos.

✅ Transmission Control Protocol/Internet Protocol (TCP/IP)

O TCP/IP é um conjunto de protocolos que permite a comunicação entre computadores na internet. O TCP garante que os dados cheguem corretamente e o IP define como os pacotes são roteados entre redes.

✅ URL e Nomes de Domínios

Uma URL (Uniform Resource Locator) identifica a localização de um recurso na internet. Os domínios facilitam a navegação, substituindo endereços IP complexos por nomes fáceis de lembrar, como "google.com".

✅ Navegadores Web

Os navegadores são programas que permitem acessar páginas da web. O primeiro navegador foi o World Wide Web (Nexus), seguido do Mosaic, Netscape, Internet Explorer, Firefox, Safari, Opera e, mais recentemente, o Google Chrome, que domina o mercado atual.
